# Диаграммы работоспособности электроприбора

## Условие

Необходимо для любого бытового прибора построить:
- Информационную модель
- Диаграмму переходов состояний (ДПС) для всех сущностей
- Модель взаимодействия объектов (МВО)
- Диаграмму последовательности действий (ДПДД) для всех действий состояний всех сущностей

Работа не предполагает написания кода - это **чисто проектировочная лабораторная**, нацеленная на отработку навыков построения различных видов диаграмм для моделирования поведения системы.

## Требования

- Выбрать любой бытовой прибор (тостер, электрочайник, стиральная машина, микроволновая печь, кофемашина и т.п.).
- Выделить все сущности прибора (например, для тостера: нагревательный элемент, механизм опускания, таймер, кнопка, пружинный механизм и т.д.).
- Для **каждой** сущности построить **ДПС** - диаграмму переходов состояний.
- Построить **МВО** - модель взаимодействия объектов.
- Построить **ДПДД** - диаграмму последовательности действий для всех действий состояний всех сущностей.
- Все диаграммы должны быть согласованы между собой и непротиворечивы.
- Каждое действие должно быть атомарным и иметь чёткое начало и конец.
- Переходы между состояниями должны быть явно обоснованы событиями.
